hexo+github博客搭建

发表于 2020-11-26

hexo+github搭建博客的过程记录，仅以自己为例。

平台：windows 10

软件安装

Node.js安装, 一路next（我也不懂那些选项做啥的），它默认添加环境变量到PATH。装完之后就可以使用npm命令了。（npm命令类似于pip命令吧）

hexo安装

  • 全局安装hexo：npm install hexo -g
  • 进入到博客存放目录：cd C:\Users\<username>\<blog-dir>
  • 初始化该文件夹：hexo init
  • 安装部署博客所需扩展：npm install hexo-deployer-git --save

至此为止，hexo相关已安装完毕

github设置

  • 创建repository，repo名为<username>.github.io，此处的username就是github账号的用户名
  • 配置ssh
    • git config --global user.name "username"
    • git config --globale user.email "email@example.com"
  • 进到 .ssh 目录：cd C:\Users\<username>\.ssh
  • 输入ssh-keygen -t rsa -C "email@example.com" 生成密钥，此处应有三个文件，实际windows上只生成了两个：id_rsa和id_rsa.pub
  • 输入ssh-keyscan -t rsa github.com >> C:\Users\<username>\.ssh\known_hosts生成known_hosts文件
  • 在github中settings添加ssh
  • 输入ssh -T git@github.com测试添加ssh是否成功

_config.yml配置

repo后是repository>code下的ssh链接

此处因为主题加载css样式需要

更换next主题

按照https://github.com/next-theme/hexo-theme-next 的 README.md

  • 在博客目录下：git clone https://github.com/next-theme/hexo-theme-next themes/next

  • 在_config.yml中更改主题：theme: next

  • 到这步似乎第一次打开主页出现了下述问题

    {% extends ‘_layout.swig‘ %} {% import ‘_macro/post.swig‘ as post_template %}

  • 之后重新装了一遍，又按照README.md输入：

    • cd themes/next
    • git tag -l
    • git checkout tags/v8.0.0
    • git checkout master

博客部署

  • 清除缓存： hexo clean
  • 静态文件生成：hexo g
  • 博客部署：hexo d

注：

  • 要先在博客目录下hexo init初始化，此后他会生成 _config.yml 配置文件。我一开始先建立了仓库，然后在settings中选择了主题（它也会生成一个 _config.yml 文件），该文件不同于前者。
